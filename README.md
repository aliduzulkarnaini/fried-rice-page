## Project Summary 

This is a responsive, single-page recipe website for Fried Rice. It presents cooking instructions, ingredients, prep time, and nutritional data in a clean, card-based layout. The page is built with pure HTML & CSS and adapts seamlessly from desktop to mobile.

## What i learnt Learnings

Semantic HTML – Using `<main>, <section>, and <table>` improves readability and SEO.

CSS Interactivity – Hover effects (transform, box-shadow) and a load animation (@keyframes) enhance user engagement.

Responsive Design – Media queries and CSS Grid (2‑column ingredients that collapse to 1‑column) ensure a great experience on any screen.

Visual Hierarchy – Warm earth tones and strategic use of background colors guide the reader’s eye naturally.

Accessibility – Focus states and proper contrast make the page usable for everyone.

Image Handling – object-fit: cover keeps the hero image proportional across devices.


### In short: I built a polished, fully responsive recipe page and deepened my skills in layout, animation, responsiveness, and design consistency — all without external libraries.
